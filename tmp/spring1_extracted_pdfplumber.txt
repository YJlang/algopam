HTTP status
1XX | Informational
요청이 수신되었고 프로세스가 계속되는 중
2XX | Success
요청이 성공적으로 처리됨
3XX | Redirection

4XX | Client Error
클라이언트의 요청이 잘못되었거나 완료할 수 없음
5XX | Server Error
서버가 유효한 요청을 처리하는 데 실패했음

Localhost & Port
IP란?:
127.0.0.1 → IP주소 (숫자)
→ 인터넷에서 컴퓨터를 구분하기 위한 ‘주소‘이다.
- 예시: 전화번호
→비유: IP = 집주소
localhost → 도메인 이름 (이름)
- 우리가 택배를 보낼 때 주소가 필요하듯이, 인터
넷에서도 데이터를 보내려면 주소가 필요함.
- 예시: 사람 이름
결국, 둘 다 내 컴퓨터 자기 자신을 가리킨다.
Port → 상세주소 (번지 수)
- 예시: 경기도 안양시 만안구 성결대학교
“성결관” 성303호
